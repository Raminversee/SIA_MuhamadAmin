# Sistem Informasi Akademik (SIA)

Repository ini merupakan hasil praktikum Mata Kuliah Web Dinamis menggunakan Express.js, TypeScript, MySQL, dan Next.js.

---

# Tugas Kelas Pertemuan 12

Materi yang dipelajari:
- Relasi tabel Mahasiswa dan Prodi (One-to-Many)
- Query JOIN antara tabel Mahasiswa dan Prodi
- Upload foto mahasiswa menggunakan Multer
- Menampilkan foto mahasiswa pada frontend Next.js
- Implementasi Search data mahasiswa
- Filter berdasarkan Program Studi
- Pagination pada backend dan frontend
- Integrasi Backend Express.js dengan Frontend Next.js

---

# Tugas Mingguan Pertemuan 12

Seluruh fitur pada tugas telah berhasil diimplementasikan dan diuji, meliputi:

- ✅ Mengubah tabel mahasiswa menggunakan `prodi_id` sebagai foreign key
